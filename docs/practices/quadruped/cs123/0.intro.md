# 从零到一搭建四足机器人

这是 [Stanford CS123 · A Hands-On Introduction to Building AI-Enabled Robots](https://cs123-stanford.readthedocs.io/en/latest/) 的中文仿真版重实现，面向自学与课程复刻。我们会用 **MuJoCo** 作为统一仿真栈，从一个单自由度执行器一路搭到一只能听懂语言指令自己走路的四足机器人。

## 课程定位

- **与原课程的差异**：原课程围绕 Stanford Pupper 实机；本课程全部章节都跑在 MuJoCo 仿真里，不依赖实体硬件，降低上手门槛。真机部分指向站点的 [SO-101 + LeRobot 真机教程](/docs/practices/robot-arm/data-collection/so101-lerobot-real)。
- **不重复基础教程**：PD、运动学、MuJoCo、RL 等概念在 [基础教程](/docs/foundations/intro) 里已经讲过。本课程把这些工具拼成一台完整的机器人，强调**连贯叙事 + 动手组装**。
- **一条明确的主线**：每章都围绕同一个 Pupper-like 四足机器人模型推进，最终产出一个能被自然语言驱动的仿真 demo。

## 你将亲手做出来的东西

1. 单关节 PD 控制可视化（第 1 章）
2. 多连杆机械臂正/逆解（第 2–3 章）
3. Pupper 四足 URDF + MuJoCo 场景（第 4 章）
4. 开环 trot 步态（第 5 章）
5. RL 训练出来的 trot 策略（第 6 章）
6. 语言指令 → 动作的端到端 demo（第 7 章）
7. 带相机的视觉追踪并接入 LLM 工具（第 8 章）

## 前置知识

- Python 基础，能看懂函数 / 类 / NumPy
- 高数里的矩阵、偏导、梯度
- 看过 [仿真与可视化 · 绪论](/docs/foundations/simulation/intro) 和 [MuJoCo 快速上手](/docs/foundations/simulation/mujoco)
- 看过 [强化学习 · 绪论](/docs/foundations/rl-for-robotics/intro)（[§6](/docs/practices/quadruped/cs123/rl-gait) 之后才用得到）

## 学习节奏（建议）

| Week | 内容 | 主要产出 |
|---|---|---|
| 1 | 0 + 1 | 跑通 MuJoCo，完成单关节 PD |
| 2 | 2 + 3 | 机械臂 FK / IK |
| 3 | 4 | Pupper URDF 在仿真里站起来 |
| 4 | 5 | 开环 trot |
| 5 | 6 | 一条 RL 训练的 trot 策略 |
| 6 | 7 + 8 | 语言控制 demo + 视觉追踪接入 |

每周建议产出：一段录屏 / 一张 reward 曲线 / 一段 100 字实验小结，归档到自己的笔记里即可。
